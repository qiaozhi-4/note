| 属性名称 (key)              | 值 (你的例子)                          | 含义解释（中文）                                                                 | 常见用途 / 注意事项                                                                 |
|-----------------------------|----------------------------------------|----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| elementId                   | 00000000-0000-004d-0000-032a00000004  | Appium 内部为该元素生成的唯一标识符（UUID 格式）                                 | 用于后续操作（如 click、sendKeys），Appium 自己的内部 ID，不是 Android 原生属性   |
| index                       | 2                                      | 该元素在同级兄弟节点中的索引（从 0 开始计数）                                    | 用于 XPath 中的 [3]（index=2 对应第 3 个）                                          |
| package                     | com.xg.gamebox                         | 元素所属 App 的包名                                                              | 确认元素来自哪个应用                                                                |
| class                       | android.widget.EditText                | Android 控件类名（这里是输入框 EditText）                                        | 用于 className 定位；这是一个可输入文本的 EditText                                 |
| text                        | ""                                     | 元素当前显示的文本内容                                                           | 空 → 目前没有可见文本（可能是 hint 提示文本，或用户还没输入）                      |
| checkable                   | false                                  | 是否是可勾选控件（如 CheckBox、Switch）                                          | false → 不是勾选框                                                                  |
| checked                     | false                                  | 是否已处于勾选/选中状态                                                          | 只对 checkable=true 的控件有意义                                                    |
| clickable                   | true                                   | 是否可以被点击（响应 onClick 事件）                                              | true → 支持 .click() 操作                                                           |
| enabled                     | true                                   | 是否启用（未禁用状态）                                                           | false 时通常呈灰色，不可交互                                                        |
| focusable                   | true                                   | 是否可以获得焦点（键盘/触屏焦点）                                                | 输入框通常为 true                                                                   |
| focused                     | false                                  | 当前是否已获得焦点                                                               | 用于判断当前焦点在哪个输入框                                                        |
| long-clickable              | true                                   | 是否支持长按（弹出上下文菜单）                                                   | true → 支持长按操作                                                                 |
| password                    | false                                  | 是否是密码输入框（内容会隐藏显示）                                               | false → 普通输入框，不是密码类型                                                    |
| scrollable                  | false                                  | 是否可滚动（如 ListView、ScrollView）                                            | false → 这个元素本身不可滚动                                                        |
| selected                    | false                                  | 是否被选中（如 Tab、RadioButton 的选中状态）                                     | 用于检查单选/多选状态                                                               |
| bounds                      | [42,624][1038,792]                     | 元素在屏幕上的绝对边界坐标（左上角 x,y → 右下角 x,y）                            | 格式：[left,top][right,bottom]，像素单位；常用于坐标点击或验证位置                  |
| displayed                   | true                                   | 是否在屏幕上可见（不被遮挡且未隐藏）                                             | true → 当前可见、可交互                                                             |
| a11y-important              | true                                   | 是否被视为无障碍（Accessibility）重要元素                                        | 通常因为有 text、hint 或 content-desc；无障碍测试中优先考虑                        |
| screen-reader-focusable     | true                                   | 是否可被屏幕阅读器（如 TalkBack）聚焦                                            | true → 支持 TalkBack 导航（无障碍友好）                                             |
| drawing-order               | 0                                      | 绘制顺序（相对于同级兄弟的 z-order，值越大越在上层）                             | 用于处理重叠控件时的点击优先级（API 24+）                                           |
| showing-hint                | false                                  | 是否正在显示 hint（提示文本，如 EditText 的 placeholder）                        | false → 当前没有显示 hint（或 hint 不显示）                                         |
| text-entry-key              | false                                  | 是否是键盘/输入法上的文本输入键（API 较高）                                      | 通常在软键盘元素上为 true；这里是普通 EditText，所以 false                          |
| dismissable                 | false                                  | 是否可 dismiss（关闭，如 Dialog、Toast、侧边栏）                                 | false → 不支持 dismiss 手势                                                         |
| a11y-focused                | false                                  | 是否当前被无障碍焦点（Accessibility focus）选中                                  | TalkBack 模式下焦点位置；false → 不是当前 TalkBack 焦点                             |
| heading                     | false                                  | 是否被标记为标题（Heading，API 28+）                                             | 用于无障碍导航（如文章/列表标题）                                                  |
| live-region                 | 0                                      | 动态内容更新模式（0=无，1=polite 礼貌读，2=assertive 强制读）                    | 0 → 非 live region（内容变化时不会自动读屏）                                        |
| context-clickable           | false                                  | 是否支持上下文点击（API 23+，如右键菜单）                                        | 很少见，通常 false                                                                  |
| content-invalid             | false                                  | 内容是否无效（表单验证失败，如输入错误）                                         | 用于无障碍提示错误状态；false → 内容有效                                            |
| window-id                   | 77                                     | 元素所属的无障碍窗口 ID（AccessibilityWindowInfo 的标识）                        | 内部使用，多窗口/弹窗/分屏场景区分窗口                                              |
