### 面试自我介绍 (Android 开发岗)

**面试官您好，我叫夏豪。**

我毕业于**湖南电子科技**，所学专业是**软件工程**。今天面试的岗位是 **Android 开发工程师**。

近期工作中，我一直专注于移动端应用开发，主要技术栈涵盖了 Java、Kotlin 以及 C++ Native 开发。

我最近负责的一个核心项目是 **GameBox**，这是一个基于 **Kotlin + C++** 混合架构的 Android 游戏辅助/工具类应用。在这个项目中，我主要负责了以下几个关键部分：

1. **现代化 UI 构建**： 项目完全采用 **Jetpack Compose (Material Design 3)** 进行 UI 开发。无论是应用的主页、详情页还是登录模块，我都使用了声明式的 UI 编程范式，结合 Kotlin Coroutines 和 Flow 进行状态管理，极大地提高了代码的可维护性和开发效率。这也是我目前最擅长和推崇的开发模式。

2. **底层 Native 交互 (JNI)**： GameBox 不仅仅是一个简单的 CRUD 应用，它涉及到深度的底层交互。我实现了 **Kotlin 与 C++ 的互调 (JNI)**，通过 

   ```
   NativeBridge
   ```

    

   模块处理高性能计算和底层逻辑。

   

3. **系统级开发调试**： 项目还涉及到了 Android 系统的**进程注入与 HOOK 技术**（基于 Xposed 框架），我处理了动态 SO 库加载、跨进程通信以及 SELinux 权限检测等具有挑战性的技术难题。

**关于在这份工作中可能涉及的技术匹配度：** 虽然我了解到目前很多存量项目可能还在使用传统的 XML 布局，而我近期主要专注于 Jetpack Compose 的实战，但我具备扎实的 Android 基础，对 View 体系也有基本的理解，能够快速上手维护现有代码，并乐于协助团队逐步向现代化的 Compose 架构迁移。

我不仅能够处理上层的业务逻辑和 UI 开发，也能深入到底层 Native 领域解决性能或底层兼容性问题。希望能有机会将我的这些经验应用到咱们公司的项目中。
