# 面试自我介绍

面试官您好，我叫夏豪，今年24岁，本科毕业于长沙理工大学。目前拥有3年的软件开发经验，主要的求职意向是开发工程师。

我的核心优势在于：**同时具备扎实的“Android底层/逆向技术”与“Web全栈开发能力”，能够胜任从系统底层到业务上层的跨领域开发工作。**

在最近的工作经历中，我有两个具有代表性的项目，分别体现了这两方面的能力：

**第一，在Android底层与原生开发方面：**
我独立设计并开发了 **GameBox** 项目。这是一个基于 Xposed/LSPosed 框架的 Android 模块，旨在为 Unity 游戏提供运行时功能增强。
在这个项目中，我采用了 **Java/Kotlin + C++ 的混合架构**。我不仅负责应用层的开发，还深入到 Native 层，利用 C++ 和 Dobby 框架实现了对图形渲染管线（如 `eglSwapBuffers`）的 Hook，并集成了 ImGui 实现高性能的自定义浮窗。
这个项目展示了我对 Android 系统机制、JNI 通信、内存操作以及 Unity 引擎（IL2CPP）底层结构的深入理解和实战能力。

**第二，在全栈应用开发方面：**
我主导开发了 **Quiz** 综合问答与任务管理平台。这是一个基于 **Java Spring Boot + Vue 3 (UniApp)** 的全栈项目。
我负责了从数据库设计、后端 API 接口实现（包括 RBAC 权限系统、JWT 认证）到前端多端适配（App、H5、小程序）的全流程开发。在该项目中，我特别设计了设备指纹验证与脚本记录功能，有效解决了业务场景中的安全风控问题。
这个项目证明了我具备构建完整业务系统、处理高并发场景以及进行前后端性能优化的能力。

**总结来说：**
我是一个既能“下沉”钻研底层技术（Hook、C++、逆向），又能“上浮”构建完整业务系统（Spring Boot、Vue）的开发者。我对技术充满热情，善于解决复杂问题，希望能利用我的技术栈为贵公司的产品研发贡献力量。
