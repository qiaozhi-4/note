##  JNI 类型签名符号列表





### 1. 基本数据类型 (Primitive Types)

| **符号** | **Java 类型** | **JNI 类型 (C/C++)** | **描述**                   |
| -------- | ------------- | -------------------- | -------------------------- |
| `Z`      | `boolean`     | `jboolean`           | 布尔值                     |
| `B`      | `byte`        | `jbyte`              | 有符号字节                 |
| `C`      | `char`        | `jchar`              | Unicode 字符               |
| `S`      | `short`       | `jshort`             | 有符号短整型               |
| `I`      | `int`         | `jint`               | 有符号整型                 |
| `J`      | `long`        | `jlong`              | 有符号长整型               |
| `F`      | `float`       | `jfloat`             | 单精度浮点数               |
| `D`      | `double`      | `jdouble`            | 双精度浮点数               |
| `V`      | `void`        | `void`               | 无返回值（仅用于方法签名） |

------



### 2. 引用类型 (Reference Types)

| **符号**              | **Java 类型** | **JNI 类型 (C/C++)** | **描述**                                                     |
| --------------------- | ------------- | -------------------- | ------------------------------------------------------------ |
| `L;`                  | 任何类类型    | `jobject` 或子类型   | 以 `L` 开头，后跟类的完整限定名（使用 `/` 而非 `.`），以 `;` 结尾。 |
| `[`                   | 任何数组类型  | `jarray` 或子类型    | 以 `[` 开头，后跟数组元素的类型签名（可以是基本类型或引用类型）。 |
| `Ljava/lang/String;`  | `String`      | `jstring`            | 字符串的特定引用类型                                         |
| `[I`                  | `int[]`       | `jintArray`          | 整型数组                                                     |
| `[Ljava/lang/Object;` | `Object[]`    | `jobjectArray`       | 对象数组                                                     |

 **注意：** 在引用类型中，如 `L;`，类名必须使用斜杠 (`/`) 作为包名分隔符，而不是点号 (`.`)。例如，`java.lang.Object` 的签名是 `Ljava/lang/Object;`。 

------



### 3. 方法签名格式



方法签名将参数和返回值组合在一起，格式为：

$$\text{(}\text{参数类型签名}_{1}\text{参数类型签名}_{2}\text{...}\text{)}\text{返回值类型签名}$$

| **部分** | **描述**                                             |
| -------- | ---------------------------------------------------- |
| `()`     | 括号内包含所有参数的类型签名，按顺序排列。           |
| 括号外   | 紧跟返回值类型签名（对于无返回值的方法，使用 `V`）。 |

 示例： 

| **Java 方法签名**                        | **JNI 签名**                               |
| ---------------------------------------- | ------------------------------------------ |
| `void myMethod()`                        | `()V`                                      |
| `int myMethod(String s)`                 | `(Ljava/lang/String;)I`                    |
| `double myMethod(int i, float f)`        | `(IF)D`                                    |
| `String myMethod(boolean[] b, Object o)` | `([ZLjava/lang/Object;)Ljava/lang/String;` |
